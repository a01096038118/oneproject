Find the SeaDronesSee Object Detection v2 Dataset here. Note that the test set annotations are witheld. You need to upload your predictions on the SeaDronesSee webserver. However, the **meta data annotations (altitude, gimbal angle, ...)** are available also for the test set (only in the Uncompressed Version though). You are encouraged use these to leverage your object detection model. Find a Faster-RCNN ResNet18 Baseline json-prediction file here (sample_submission_odv2.json); this file could be uploaded to the Object Detection v2 track right away.

As the images are mostly of 4K resolution and png, they are of large size. Find a jpeg-compressed version in "Compressed Version". Note that for the submission of your prediction, your json-submission needs to have the .png-file names, **not** the .jpg-names.
